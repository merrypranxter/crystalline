# 03 — Tetragonal Crystal System

## Overview

The **tetragonal** system has one unique 4-fold (or 4̄-fold) rotation axis — the c-axis.
Two equal axes (a = b) lie in the basal plane perpendicular to c.
The unit cell is a square prism (rectangular box with square cross-section).

---

## Lattice Parameters

| Parameter | Constraint | Notes |
|-----------|-----------|-------|
| a | free | Basal axis |
| b | = a | Equal to a by 4-fold symmetry |
| c | free (≠ a) | Unique (4-fold) axis |
| α | = 90° | |
| β | = 90° | |
| γ | = 90° | |

**Degrees of freedom:** 2 (a and c)

---

## Bravais Lattices

| Symbol | Name | Points/Cell |
|--------|------|------------|
| tP | Primitive tetragonal | 1 |
| tI | Body-centered tetragonal | 2 |

The body-centered tetragonal (BCT) lattice becomes BCC when c/a = 1.
Indium metal (tI, c/a = 1.521) is a classic example of distorted BCC.

## Mathematical Formulas

### d-Spacing
```
1/d²_hkl = (h² + k²)/a² + l²/c²
```

### Unit Cell Volume
```
V = a² · c
```

### Tetragonal Structure: Rutile (TiO₂)
- Ti at (0,0,0) and (½,½,½)
- O at ±(u,u,0) and ±(½+u,½-u,½) with u ≈ 0.305
- TiO₆ octahedra share edges along c and corners in ab-plane

### Tetragonal Structure: Zircon (ZrSiO₄)
- Space group I4₁/amd (No. 141)
- Zr in triangular dodecahedra (CN=8), Si in isolated tetrahedra
- Building block of many REE accessory minerals (monazite, xenotime)

---

## Visual Characteristics

- **Crystal habit:** Square prisms, tetragonal bipyramids, combinations
- **Sections:** Square cross-sections ⊥ c; rectangular sections ∥ c
- **Cleavage:** Prismatic {110} or {100}, pyramidal {011}
- **Optical appearance:** Uniaxial — similar to hexagonal but with 4-fold symmetry
- **Common twinning:** Rutile knee twins, chalcopyrite mimics cubic pyrite
