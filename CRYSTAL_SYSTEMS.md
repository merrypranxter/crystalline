# Crystal Systems — Comprehensive Overview of All Seven Systems

Crystallography divides all crystalline solids into seven crystal systems based on the
symmetry of the unit cell — the smallest repeating unit of the crystal lattice.
Each system is defined by the relationships between the three axial lengths (a, b, c)
and the three interaxial angles (α, β, γ).

## 1. Cubic (Isometric) System

The highest-symmetry crystal system. The unit cell is a perfect cube.

**Defining constraint:** a = b = c ; α = β = γ = 90°

**Characteristic symmetry:** Four 3-fold rotation axes along ⟨111⟩ body diagonals

**Bravais lattices (3):**
| Symbol | Name | Lattice Points/Cell |
|--------|------|-------------------|
| cP | Primitive | 1 |
| cI | Body-centered | 2 |
| cF | Face-centered | 4 |

**Point groups (5):**
| Symbol | Schönflies | Order | Description |
|--------|-----------|-------|-------------|
| 23 | T | 12 | Chiral tetrahedral |
| m3̄ | Th | 24 | Pyritohedral |
| 432 | O | 24 | Chiral octahedral |
| 4̄3m | Td | 24 | Tetrahedral |
| m3̄m | Oh | 48 | Full octahedral (holohedric) |

**Key space groups:** Fm3̄m (225), Im3̄m (229), Pm3̄m (221), Fd3̄m (227), Ia3̄d (230),
Pa3̄ (205), F4̄3m (216), P4̄3n (218)

**Packing fractions and coordination:**
| Structure | Packing | CN | Example |
|-----------|---------|-----|---------|
| Simple cubic (cP) | 52.4% | 6 | Polonium |
| Body-centered (cI) | 68.0% | 8 | Iron (α), W, Mo |
| Face-centered (cF) | 74.05% | 12 | Cu, Au, Ag, Al |
| Diamond cubic | 34.0% | 4 | Diamond, Si, Ge |

**Optical property:** All cubic crystals are **isotropic** — single refractive index, zero birefringence.

---

## 2. Hexagonal System

One unique 6-fold symmetry axis (the c-axis) perpendicular to a hexagonal basal plane.

**Defining constraint:** a = b ≠ c ; α = β = 90° ; γ = 120°

**Bravais lattices (1):** hP (Primitive hexagonal only)

**Point groups (7):**
| Symbol | Schönflies | Description |
|--------|-----------|-------------|
| 6 | C₆ | Hexagonal rotation |
| 6̄ | C₃h | Hexagonal rotoinversion |
| 6/m | C₆h | Hexagonal with mirror ⊥ c |
| 622 | D₆ | Hexagonal dihedral |
| 6mm | C₆v | Hexagonal pyramidal |
| 6̄m2 | D₃h | Ditrigonal dipyramidal |
| 6/mmm | D₆h | Dihexagonal dipyramidal (holohedric) |

**Key space groups:** P6/mmm (191), P63/mmc (194), P63mc (186), P6̄c2 (190),
P63/m (176), P6₃ (173), P6₂22 (180)

**c/a ratio:** The ideal HCP ratio is **c/a = √(8/3) ≈ 1.633**
- Mg: 1.624 (near ideal)    - Zn: 1.856 (elongated)
- Be: 1.568 (compressed)    - Ti: 1.587 (compressed)

**Optical property:** **Uniaxial** — two principal refractive indices (no, ne).
Positive uniaxial: ne > no. Negative uniaxial: ne < no.

---

## 3. Tetragonal System

One unique 4-fold symmetry axis (c-axis); two equal axes perpendicular to it.

**Defining constraint:** a = b ≠ c ; α = β = γ = 90°

**Bravais lattices (2):** tP (Primitive), tI (Body-centered)

**Point groups (7):**
| Symbol | Schönflies | Description |
|--------|-----------|-------------|
| 4 | C₄ | Tetragonal rotation |
| 4̄ | S₄ | Tetragonal rotoinversion |
| 4/m | C₄h | With mirror ⊥ c |
| 422 | D₄ | Tetragonal dihedral |
| 4mm | C₄v | Ditetragonal pyramidal |
| 4̄2m | D₂d | Tetragonal scalenohedron |
| 4/mmm | D₄h | Ditetragonal dipyramidal (holohedric) |

**Key space groups:** P4₂/mnm (136, rutile type), I4₁/amd (141, zircon type),
I4₁/a (88, scheelite type), I4̄2d (122), P4/mnc (128)

**Optical property:** **Uniaxial** (same as hexagonal, different symmetry).

---

## 4. Trigonal (Rhombohedral) System

A subgroup of the hexagonal system — one 3-fold symmetry axis.
Can be described in **hexagonal axes** (standard) or **rhombohedral axes**.

**Defining constraint (hex axes):** a = b ≠ c ; α = β = 90° ; γ = 120°
**Defining constraint (rhomb axes):** a = b = c ; α = β = γ ≠ 90°

**Bravais lattices (2):** hP (hexagonal primitive), hR (rhombohedral)

**Point groups (5):**
| Symbol | Schönflies | Description |
|--------|-----------|-------------|
| 3 | C₃ | Trigonal rotation |
| 3̄ | S₆ (C₃i) | Trigonal rotoinversion |
| 32 | D₃ | Trigonal dihedral |
| 3m | C₃v | Ditrigonal pyramidal |
| 3̄m | D₃d | Ditrigonal scalenohedral (holohedric) |

**Key space groups:** R3̄c (167, calcite type), R3̄ (148), R3c (161),
R3m (160), P3₁21 (152, quartz type), P3₁12 (151), P3̄m1 (164)

**Optical property:** **Uniaxial**. Calcite has extreme birefringence (δ = 0.172).

---

## 5. Orthorhombic System

Three mutually perpendicular axes of **all different** lengths; all angles 90°.

**Defining constraint:** a ≠ b ≠ c ; α = β = γ = 90°
(Convention: a < b < c)

**Bravais lattices (4):** oP, oC (oA/oB), oI, oF

**Point groups (3):**
| Symbol | Schönflies | Description |
|--------|-----------|-------------|
| 222 | D₂ | Three 2-fold axes |
| mm2 | C₂v | Two mirrors + 2-fold axis |
| mmm | D₂h | Three mirrors + inversion (holohedric) |

**Key space groups:** Pnma (62, most common), Cmca (64), Fddd (70),
Pbnm (62 setting), Fdd2 (43), C2221 (20)

**Optical property:** **Biaxial** — three principal refractive indices (nx, ny, nz).

---

## 6. Monoclinic System

Three axes of **all different** lengths; one oblique angle (β ≠ 90°).

**Defining constraint:** a ≠ b ≠ c ; α = γ = 90° ; β ≠ 90°
(Convention: b is the unique 2-fold axis; β > 90°)

**Bravais lattices (2):** mP (Primitive), mC (C-centered, also written mA or mI)

**Point groups (3):**
| Symbol | Schönflies | Description |
|--------|-----------|-------------|
| 2 | C₂ | Monoclinic rotation |
| m | Cs | Single mirror plane |
| 2/m | C₂h | Monoclinic prismatic (holohedric) |

**Key space groups:** P2₁/c (14, most common of all!), C2/c (15),
C2/m (12), P2₁ (4), Cm (8)

Space group P2₁/c (No. 14) contains **more known crystal structures** than
any other space group — roughly 35% of all organic crystal structures.

**Optical property:** **Biaxial** — complex optical orientation with β = Bxa or Bxo.

---

## 7. Triclinic System

The least symmetric system. No required angular relationships.

**Defining constraint:** a ≠ b ≠ c ; α ≠ β ≠ γ ≠ 90°

**Bravais lattices (1):** aP (Primitive only)

**Point groups (2):**
| Symbol | Schönflies | Description |
|--------|-----------|-------------|
| 1 | C₁ | No symmetry (chiral) |
| 1̄ | Cᵢ | Inversion center only (holohedric) |

**Space groups:** Only P1 (No. 1) and P1̄ (No. 2)

Despite minimal symmetry, triclinic minerals include important feldspar group members
(albite, anorthite, microcline) that make up ~60% of Earth's crust.

**Optical property:** **Biaxial** — most complex optical behavior, 2V varies widely.
